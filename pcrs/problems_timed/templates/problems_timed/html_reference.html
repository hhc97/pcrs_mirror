Bolded text:
<b>text here</b>
e.g. text here

Italicized text:
<i>text here</i>
e.g. text here

Underlined text:
<u>text here</u>
e.g. text here

Crossed out text:
<del>text here</del>
e.g. text here
Modify the size, color, and/or alignment of text:
<font size='24' color='red' align=' right'>text here</font>
* Note: size is an integer, color is a color, and align is left/center/right.

Insert image:
<img src='url-to-image'>

Insert image and set size:
<img src='url-to-image' width='100' height='50'>

Insert YouTube video:
* While at a video, click on "share" then "embed", and copy the code given.
* It will look something like this:
<iframe width='560' height='315' src='url-to-video' frameborder='0' allowfullscreen></iframe>
